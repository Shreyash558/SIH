# **App Name**: Learnscape

## Core Features:

- Personalized Dashboard: Display key educational metrics for students, like completed lessons and current courses.
- Learning Path Tool: Suggest the next most relevant educational module for each student. This tool considers the student's progress, learning style, and the curriculum structure to provide personalized recommendations.
- Interactive Lessons: Lessons can be presented as text, video, or quizzes. Show only interactive lessons on this page.
- Offline Content: Download content (lessons, quizzes) for offline access.
- Multilingual Interface: Enable selection of content based on the user's choice of language.

## Style Guidelines:

- Background color: Dark navy blue (#1A237E), offering a professional and calm aesthetic.  This choice conveys depth and trustworthiness while maintaining visual accessibility. This color works particularly well with lighter text and graphic elements, enhancing readability and focus.
- Primary color: Light purple (#9FA8DA), a vibrant hue that creates an optimistic impression without overwhelming the eye. Because it is much lighter than the background color, it allows for very strong contrast.
- Accent color: Muted lavender (#D5C4E6) to provide subtle contrast.
- Font: 'PT Sans', sans-serif, for body text and headings.
- Dashboard cards with rounded corners.
- Simple, outline-style icons to match the user's example.
